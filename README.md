# 🎬 Hotstar Content Analysis Dashboard - Power BI

## 📌 Project Overview
This project is an interactive **Hotstar Content Analysis Dashboard** built using Power BI.  
It provides insights into content distribution, genre popularity, and streaming trends.

---

## 🎯 Objectives
- Analyze content trends on Hotstar
- Identify popular genres and categories
- Understand content distribution by year
- Explore ratings and audience targeting

---

## 📂 Dataset Features
The dataset includes:
- Hotstar ID
- Title
- Description
- Genre
- Release Year
- Age Rating
- Running Time
- Seasons (for series)

---

## 📊 Dashboard Insights
Key insights covered:
- 🎭 Genre-wise Content Distribution
- 📅 Year-wise Content Trends
- 🔞 Age Rating Analysis
- ⏱️ Running Time Insights
- 📺 Movies vs Series Comparison

---

## 🛠️ Tools & Technologies Used
- Power BI
- Power Query (Data Transformation)
- DAX (Data Analysis Expressions)

---

## 🚀 Features

- Clean and visually appealing design
- Dynamic charts and visuals
- Easy-to-understand insights
